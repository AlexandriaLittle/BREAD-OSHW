# BREAD - Boradly Reconfigurable and Expandable Automation Device

***Please check the 1.Documentation folder for the original paper and other related documents, licenses, etc!***

The original project documentation is published in the open-access journal Hardware X 15 and is authored by Shane Oberloier, Nicholas G. Whisman, Finn Hafting, and Joshua M. Pearce. Other contributors have added to the body of work since. The documentation can be found online at the Center for Open Science's website (osf.io). Search 'Bread Slice' and you will find plenty!
